## 🚀 Deploy in 3 Steps (15 Minutes Total)

### Step 1: Push to GitHub (5 minutes)

Open terminal in `law-firm-web-app` folder:

```bash
# Remove old git
rm -rf .git

# Initialize new repository
git init
git add .
git commit -m "Initial commit - Law firm website"
git branch -M main

# Connect to your GitHub
git remote add origin https://github.com/unknowncoder84/darshana-maam.git

# Push code
git push -u origin main
```

✅ **Done!** Code is on GitHub

---

### Step 2: Deploy to Netlify (5 minutes)

1. Go to https://app.netlify.com
2. Click "Add new site" → "Import an existing project"
3. Choose "Deploy with GitHub"
4. Select repository: `darshana-maam`
5. Build settings (auto-detected):
   - Build command: `npm run build`
   - Publish directory: `.next`
6. Click "Deploy site"
7. Wait 2-3 minutes

✅ **Done!** Website is live!

---
