# Admin Dashboard User Guide

## Quick Start Guide

### Accessing the Admin Dashboard

1. **Find the Admin Login Link**
   - Scroll to the bottom of any page on the website
   - Look for the "Admin Login" link with a lock icon (🔒) in the footer
   - Click the link to access the login page

2. **Login**
   - Enter your username
   - Enter your password (minimum 8 characters alphanumeric)
   - Click "Sign In"
   - You'll be redirected to the dashboard

### Dashboard Overview

After logging in, you'll see:
- **Welcome message** with your username and role
- **Statistics cards** showing content counts
- **Calendar widget** (if you're an admin) with upcoming events
- **Quick actions** for common tasks
- **Sidebar navigation** for accessing different sections

## Features by Role

### Admin Role (Full Access)

Admins have access to all features:

#### 1. Dashboard
- View statistics for all content types
- See calendar with court dates and appointments
- Access quick actions

#### 2. News Management
- Create, edit, and delete news items
- Upload featured images
- Publish/unpublish news
- View all news items

#### 3. Articles Management
- Create, edit, and delete articles
- Add tags for categorization
- Set SEO metadata
- Upload featured images
- Publish/unpublish articles

#### 4. Videos Management
- Create, edit, and delete videos
- Upload video files or add YouTube URLs
- Add thumbnails
- Publish/unpublish videos

#### 5. User Management (Admin Only)
- View all system users
- Create new users with username, password, and role
- Edit user roles
- Delete users (except yourself)

#### 6. Firm Settings (Admin Only)
- Update firm name
- Update contact information
- Update social media links
- Manage firm-wide settings

### User Role (Content Management)

Users have access to content management only:

#### 1. Dashboard
- View statistics for all content types
- Access quick actions for content creation

#### 2. News Management
- Create, edit, and delete news items
- Upload featured images
- Publish/unpublish news

#### 3. Articles Management
- Create, edit, and delete articles
- Add tags and SEO metadata
- Upload featured images
- Publish/unpublish articles

#### 4. Videos Management
- Create, edit, and delete videos
- Upload videos or add YouTube URLs
- Add thumbnails
- Publish/unpublish videos

**Note**: Users cannot access User Management or Firm Settings.

## Common Tasks

### Creating a New User (Admin Only)

1. Click "Users" in the sidebar navigation
2. Click the "Create User" button
3. Fill in the form:
   - **Username**: Unique username for the user
   - **Password**: Minimum 8 characters alphanumeric
   - **Role**: Select "Admin" or "User"
4. Click "Create User"
5. The new user can now log in with their credentials

### Creating a News Item

1. Click "News" in the sidebar navigation
2. Click the "Create News" button or use Quick Actions
3. Fill in the form:
   - **Title**: News headline
   - **Content**: News body (use rich text editor)
   - **Image**: Upload a featured image (optional)
   - **Published**: Toggle to make visible on public website
4. Click "Save"

### Creating an Article

1. Click "Articles" in the sidebar navigation
2. Click the "Create Article" button or use Quick Actions
3. Fill in the form:
   - **Title**: Article headline
   - **Content**: Article body (use rich text editor)
   - **Tags**: Add categories (e.g., "Cyber Crimes", "Criminal Law")
   - **Image**: Upload a featured image (optional)
   - **SEO Meta**: Add custom title, description, keywords
   - **Published**: Toggle to make visible on public website
4. Click "Save"

### Publishing/Unpublishing Content

1. Navigate to the content type (News, Articles, or Videos)
2. Find the item you want to publish/unpublish
3. Click "Edit"
4. Toggle the "Published" checkbox
5. Click "Save"

**Note**: Only published content is visible on the public website.

### Editing User Roles (Admin Only)

1. Click "Users" in the sidebar navigation
2. Find the user you want to edit
3. Click the "Edit" button
4. Change the role to "Admin" or "User"
5. Click "Save"

### Deleting a User (Admin Only)

1. Click "Users" in the sidebar navigation
2. Find the user you want to delete
3. Click the "Delete" button
4. Confirm the deletion in the dialog
5. The user will be removed from the system

**Note**: You cannot delete your own account.

## Calendar Widget (Admin Only)

The calendar widget helps you track important dates:

### Understanding the Calendar

- **Current date**: Highlighted in blue
- **Event indicators**: Small colored dots at the bottom of dates
  - 🔴 Red: Court dates
  - 🔵 Blue: Meetings
  - 🟡 Amber: Deadlines

### Navigating the Calendar

- Click the **left arrow** to go to the previous month
- Click the **right arrow** to go to the next month
- The current month and year are displayed at the top

### Upcoming Events

Below the calendar, you'll see a list of the next 3 upcoming events with:
- Event title
- Event date
- Color-coded indicator

**Note**: Calendar event creation/editing will be added in a future update.

## Tips & Best Practices

### Content Creation

1. **Use descriptive titles**: Make titles clear and informative
2. **Add images**: Featured images make content more engaging
3. **Use tags**: Categorize articles for better organization
4. **Preview before publishing**: Check content before making it public
5. **SEO metadata**: Add custom SEO data for better search rankings

### User Management

1. **Strong passwords**: Require users to use strong passwords
2. **Appropriate roles**: Assign roles based on responsibilities
3. **Regular review**: Periodically review user list and remove inactive users
4. **Document access**: Keep a record of who has access

### Security

1. **Log out**: Always log out when finished
2. **Don't share credentials**: Each user should have their own account
3. **Change passwords**: Update passwords regularly
4. **Report issues**: Contact the administrator if you notice anything unusual

## Troubleshooting

### Can't Log In

- **Check credentials**: Ensure username and password are correct
- **Password requirements**: Password must be at least 8 characters alphanumeric
- **Contact admin**: If you forgot your password, contact an administrator

### Can't Access a Feature

- **Check role**: Some features are admin-only
- **Refresh page**: Try refreshing the browser
- **Clear cache**: Clear browser cache and cookies
- **Contact admin**: If the issue persists, contact an administrator

### Content Not Showing on Public Website

- **Check published status**: Ensure the "Published" toggle is ON
- **Refresh public site**: Clear browser cache and refresh
- **Wait a moment**: Changes may take a few seconds to appear

### Image Upload Issues

- **File size**: Ensure image is not too large (max 5MB recommended)
- **File format**: Use JPG, PNG, or WebP formats
- **Internet connection**: Check your internet connection
- **Try again**: Refresh and try uploading again
